# TaskPad

Tool for task management in Haskell

## Generate task file

## CLI

```
$ taskpad --help
taskpad - operate tasks

Usage: taskpad [-v|--verbose] [-c|--config PATH] COMMAND [--version]

Available options:
  -v,--verbose             Enable verbose mode: verbosity level "debug"
  -c,--config PATH         Configuration file
  --version                Show version
  -h,--help                Show this help text

Available commands:
  new                      Create a new task file
  update                   Update task
  done                     Check done task
  tasks                    Show Tasks
  template                 Dump default config
```
